### Tools

- Typescript
- MongoDB
- Express
- Nextjs 14
- Redux
- Material UI 5


### Getting started

1. Copy `env` secret files
```
 cd backend &&  cd frontend

    touch env

    copy env.example to env

```

2. Run the App
```
 cd backend &&  cd frontend

    npm install

    npm start

```
3. View the App locally

```
 frontend : http://localhost:3000

 backend : http://localhost:4000
 
```
